# Rust LeetCode Journey 🦀🚀

Este repositorio está dedicado a resolver problemas de **LeetCode** utilizando **Rust**, con el objetivo de fortalecer mis habilidades técnicas, entender el sistema de tipos y la gestión de memoria (ownership/borrowing), y explorar soluciones creativas.

## 🎯 Objetivos
- **Dominio de Rust:** Aprender a escribir código idiomático y eficiente.
- **Pensamiento Algorítmico:** Resolver retos de diversa complejidad (Easy, Medium, Hard).
- **Calidad de Código:** Aplicar principios de diseño y buenas prácticas.
- **Aprendizaje Continuo:** Documentar el "porqué" de las soluciones más complejas.

## 🛠️ Estructura del Proyecto
Los problemas se organizan por su número identificador y nombre.
